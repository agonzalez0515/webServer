# Web Server  [![Build Status](https://travis-ci.org/agonzalez0515/webServer.svg?branch=master)](https://travis-ci.org/agonzalez0515/webServer)



This project uses:
* Gradle 6
* Java 13
* Junit 4
* Mockito
* Deployed to Heroku

### Run Server Locally on port 5000

```gradle run```

### Run unit tests

```gradle test```
